# Arachne

A project to save money for analytic workloads in cloud systems. The
inter\_query\_analysis module contains algorithms, data, and graphing scripts to
produce inter-query plans. The intra\_query\_analysis folder contains the Java
project to find, build, and execute hybrid query plans in Google BigQuery and
Amazon Redshift. 

This repository contains all scripts, algorithms, and data needed for this
analysis, but is currently very disorganized and is being made public for the
sake of paper submission. It will be ready soon for use, so keep an eye out!
